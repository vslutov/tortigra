## Installation guide ##

1. Install [nodejs](http://nodejs.org/)
2. Clone repository
3. Run `npm install` from folder with code
4. Run `cake build:all`


## Why Tortigra? ##

Tortigra - is simple way to copy files with folder hierarchy to portable device.
